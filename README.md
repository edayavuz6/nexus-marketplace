## вңЁ Features

### рҹӣҚпёҸ Shopping Experience
- **Real API Integration** вҖ” Products fetched live from [DummyJSON](https://dummyjson.com/products) with dynamic loading
- **Product Search** вҖ” Live search with 400ms debounce for performance; searches across name, brand and tags
- **Category Filtering** вҖ” Dynamic category tabs fetched from API; click any category to instantly filter
- **Smart Sorting** вҖ” Sort by Featured, Price (LowвҶ’High / HighвҶ’Low), Top Rated, or Best Deals
- **Product Detail Modal** вҖ” Full-screen modal with image gallery, specs, tag pills, quantity selector, stock info and reviews
- **Quick Add** вҖ” Hover over any product card to reveal a quick-add button without opening the modal

### рҹӣ’ Shopping Cart
- **Persistent Cart** вҖ” Cart state saved to `localStorage`; survives page refreshes and browser restarts
- **Quantity Controls** вҖ” Increment / decrement / remove individual items directly from the cart drawer
- **Subtotal & Shipping** вҖ” Automatic shipping calculation; free shipping unlocked above $99
- **Clear Cart** вҖ” One-click full cart reset
- **Slide-In Drawer** вҖ” Smooth cart panel slides in from the right with backdrop blur overlay

### вқӨпёҸ Wishlist
- **Add to Wishlist** вҖ” Heart icon on every product card; persisted to `localStorage`
- **Visual Feedback** вҖ” Active wishlist items highlighted with red heart icon
- **Accessible** вҖ” Works from both product cards and the product detail modal

### рҹ”” Toast Notifications
- **Add to Cart Toast** вҖ” Bottom-right popup with product thumbnail, name, and confirmation message
- **Auto-dismiss** вҖ” Toasts disappear automatically after 3.5 seconds
- **Manual close** вҖ” Each toast has an Г— button for immediate dismissal
- **Stacking** вҖ” Multiple toasts stack gracefully

### рҹҢҷ Dark / Light Mode
- **System-Aware Default** вҖ” Defaults to dark mode; toggle instantly in the navbar
- **Smooth Transitions** вҖ” All colors transition at `0.4s ease` вҖ” no jarring flashes
- **Full Coverage** вҖ” Every component (hero, products, cart, modal, footer) responds to the theme

### рҹҺ  Hero Carousel
- **Auto-Sliding** вҖ” Cycles through 4 premium product showcases every 5.5 seconds
- **Manual Dots** вҖ” Click the dot indicators to jump to any slide
- **Slide Counter** вҖ” Editorial `01 / 04` counter in the bottom-right corner
- **Glassmorphism Card** вҖ” Product displayed inside a frosted-glass card with gradient border and floating feature badges
- **Smooth Transitions** вҖ” Fade + scale animation on slide change; background glow color morphs per product

### рҹ–јпёҸ Product Cards
- **Hover Lift** вҖ” Cards elevate with `translateY(-6px) scale(1.01)` on hover
- **Image Zoom** вҖ” Product thumbnail zooms subtly on hover
- **Error Fallback** вҖ” If an image fails to load, a clean placeholder with category label is shown
- **Discount Badges** вҖ” Red percentage badge auto-calculated from `discountPercentage`
- **Low Stock Warning** вҖ” Amber badge shown when `stock < 10`
- **Star Ratings** вҖ” SVG star icons rendered from API rating data

### рҹ”Қ Search Modal
- **Instant Results** вҖ” Search results appear as you type (debounced at 400ms)
- **Skeleton Loaders** вҖ” Shimmer placeholders shown while results load
- **Popular Searches** вҖ” Pill buttons for trending terms when no query is entered
- **Click to Add** вҖ” Click any result to add it to cart and close the modal

### вҡЎ Performance & Code Quality
- **Custom Hooks** вҖ” `useProducts`, `useCategories`, `useProduct` abstract all API logic
- **Context API** вҖ” `CartContext` and `ToastContext` for clean global state management
- **Debounced Search** вҖ” Prevents excessive API calls on keystroke
- **Skeleton Loaders** вҖ” Full shimmer skeleton grid shown while products load
- **Error States** вҖ” Graceful error UI if the API fails
- **Semantic HTML** вҖ” Proper use of `<nav>`, `<section>`, `<main>`, `<footer>`, `<button>`
- **No External UI Library** вҖ” 100% custom components; no component library dependency

## рҹҺЁ Design System

### Color Palette

| Token | Dark Mode | Light Mode |
|---|---|---|
| Background | `#0A0A0A` | `#F8F8F8` |
| Surface | `#111111` | `#FFFFFF` |
| Surface Alt | `#1A1A1A` | `#F0F0F0` |
| Text Primary | `#F5F5F5` | `#0A0A0A` |
| Text Muted | `rgba(245,245,245,0.45)` | `rgba(10,10,10,0.45)` |
| Accent | `#6366F1` (Indigo) | same |
| Accent 2 | `#8B5CF6` (Violet) | same |
| Success | `#10B981` | same |
| Danger | `#EF4444` | same |
| Gold | `#F59E0B` | same |

### Typography

- **Display / Headings** вҖ” [Syne](https://fonts.google.com/specimen/Syne) вҖ” weight 600вҖ“800
- **Body / UI** вҖ” [Inter](https://fonts.google.com/specimen/Inter) вҖ” weight 300вҖ“600

### Key Design Patterns

- **Glassmorphism** вҖ” `backdrop-filter: blur(20px)` + `rgba` backgrounds on modals, hero card, and overlays
- **Gradient Accents** вҖ” `linear-gradient(135deg, #6366F1, #8B5CF6)` on CTAs, badges, and avatar initials
- **Micro-interactions** вҖ” Hover lifts, scale transforms, and opacity transitions on all interactive elements
- **Radial Glow Blobs** вҖ” Large filtered blobs behind the hero section that transition color per product slide

## рҹ”Ң API Reference

This project uses the free [DummyJSON](https://dummyjson.com) REST API.

| Endpoint | Used For |
|---|---|
| `GET /products?limit=30` | All products grid |
| `GET /products/search?q={query}` | Live search results |
| `GET /products/category/{slug}` | Category filtering |
| `GET /products/categories` | Category list in nav & grid |
| `GET /products/{id}` | Single product (available via `useProduct` hook) |
